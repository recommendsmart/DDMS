INTRODUCTION
------------

 * This module provides you with blocks of a simple calculator, emi calculator and home loan calculator.

INSTALLATION
------------
 * Install as you would normally install a contributed drupal module and enable the Simple Calculator module.

CONFIGURATION:
--------------
1. Visit /admin/structure/block and configure "Simple Calculator", "EMI Calculator", and "Home Loan Calculator" block.
2 .This block will enable Simple Calculator, EMI calculator, Home loan Calculator.
3. You can also print EMI view also.
